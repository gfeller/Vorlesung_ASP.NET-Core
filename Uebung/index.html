WED3 - Canvas

Ziel

Anhand eines BMI-Rechners sollen die im Unterricht erworbenen ASP.NET Grundlagen angewendet und vertieft werden.

Aufgabe 1

Der BMI-Rechner könnte in einer vereinfachten Variante wie folgt aussehen:

Gehen Sie bei Ihrer Implementation wie folgt vor:

  1. Erstellen Sie ein neues ASP.NET Projekt.
    • Nutzen Sie das „Web Applikation Template“ (ohne Security).
    • Testen Sie jeden Schritt indem Sie Ihr Projekt im Browser ausführen.
  3. Erstellen Sie innerhalb der Index-Seite das HTML sowie den Razor-Code für den Rechner.
  4. Erstellen Sie eine Action, welche den BMI anhand des Gewichtiges und der Höhe berechnen kann.
    • Geben Sie den BMI als Content() zurück.
    • Definieren Sie eine Klasse um diese Daten zu speichern.
    • Bei fehlerhaften Eingaben soll "Fehlerhafte Daten“ ausgegeben werden.
    • Rufen Sie diese Action von der Rechner-Form auf.
  6. Nutzen Sie die AJAX Library um das Resultat der Action direkt unterhalb des Rechners darzustellen.
  7. Definieren Sie einen BmiService, welcher die Berechnung übernimmt.
  8. Registrieren Sie den BmiService im Dependency-Injection-Container.
    • Lassen Sie sich den Service im Controller „Injecten“ und nutzen Sie die vom Dependency-Injection-Container zur Verfügung gestellte Instanz.

Aufgabe 2

Erstellen Sie eine weitere Variante des Rechners. Diese soll mit einer Razor View, welche den BMI anzeigt und die jeweilige „Stufe“ vom BMI hervorhebt, realisiert werden. Generieren Sie die BMI-Tabelle (siehe Bild) aus einem strukturierten Datenbestand. Als Basis könnte folgende Struktur dienen:

var data = new[]
{
     new {Text = "starkes Untergewicht", Max=16.0},
     new {Text = "mäßiges Untergewicht", Max=17.0},
     new {Text = "leichtes Untergewicht", Max=18.5},
     new {Text = "Normalgewicht", Max=25.0},
     new {Text = "Präadipositas", Max=30.0},
     new {Text = "Adipositas Grad I", Max=35.0},
     new {Text = "Adipositas Grad II", Max=40.0},
     new {Text = "Adipositas Grad III", Max=999.0},
};
  1. Definieren Sie eine weitere Action.
  2. Definieren Sie eine Razor-View, welche den BMI Wert entgegen nimmt und diesen hervorhebt.
  3. Definieren Sie die Daten als Array und „rendern“ Sie die Tabelle mit einer foreach-Schleife.
  4. Verbinden Sie Ihre Lösung von Aufgabe 1 und Aufgabe 2: Beim Resultat bei Aufgabe 1 soll nicht mehr nur der Content() stehen sondern die gerenderte Razor View von Aufgabe 2.
